Typora+PicGo+SMMS打造个人图床

Typora+PicGo+SMMS打造个人图床

工具

  • PicGo (图床工具)

  • Typora（markdown编辑器）

  • SM.MS （图片储存及分享服务）

步骤

1.SM.MS

地址：https://sm.ms/

注册成功会有免费的5G存储空间，够用几年了。

进入仪表盘，选择API Token，生成token，等会就要用到。

2.PicGo下载

PicGo官方解释: 一个用于快速上传图片并获取图片 URL 链接的工具

下载安装推荐镜像，比较快速。

下载源 地址/安装方式 平台
GitHub Release https://github.com/Molunerfinn/PicGo/releases All
腾讯云COS https://github.com/Molunerfinn/PicGo/releases All
山东大学镜像站 https://mirrors.sdu.edu.cn/github-release/Molunerfinn_PicGo All

安装完成，打开应用界面如下。

下载插件，搜索smms-user

接着进入图床设置，将token复制进来，备用上传域名默认smms.app，点击确认保存即可。

3.Typora

地址：https://typoraio.cn/

安装完毕后，打开文件-偏好设置-图像，按照如下图片对应勾选即可。

4.测试是否成功

  • 复制图片到Markdown文档
  • 检查SM.MS是否有上传的图片即可。

该文章用以上技术Typera、PicGo、SM.MS作成。
